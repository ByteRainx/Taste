## Running Your Own Analysis

```bash
# Analyze any researcher
taste --researcher "Researcher Name"

# Save to file
taste --researcher "Researcher Name" --output_file output.md

# Use Semantic Scholar author ID directly
taste --researcher_id 12345678 --output_file output.md
```
